# 🧬 Molecular Playground

A hands-on demo exploring protein structures and small-molecule drugs using
Pi-Science's molecular viewers, Python analysis toolkit, and MCP literature search.

## What's Inside

| File | Description |
|------|-------------|
| `data/1LYS.pdb` | Hen egg-white lysozyme (X-ray, 34 residues) — classic enzyme |
| `data/aspirin.xyz` | Aspirin (acetylsalicylic acid, C9H8O4) — the first synthetic drug |
| `data/caffeine.xyz` | Caffeine (C8H10N4O2) — most consumed psychoactive |
| `drug_properties.py` | Drug-likeness analysis (Lipinski's Rule of 5) with scatter plot |
| `analyze_binding.py` | Lysozyme secondary structure prediction |

## Try These Prompts

### Visual Exploration
- "Open data/1LYS.pdb in the molecule viewer and explore its structure."
- "View aspirin.xyz — what does the molecule look like in 3D?"

### Data Analysis
- "Run drug_properties.py and explain which drugs pass Lipinski's rules."
- "Run analyze_binding.py to analyze the lysozyme structure."

### Literature Search
- "Search for recent papers about lysozyme catalytic mechanism."
- "Find papers about aspirin's COX inhibition binding mode."
